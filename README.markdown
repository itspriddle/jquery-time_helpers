# jQuery Time Helpers

This includes some random time functions for use with
[jQuery](http://jquery.com).

## toTimestamp

`$.toTimestamp(seconds)` converts seconds to a timestamp (mm:ss).

Examples:

    $.toTimestamp(30);  // 00:30
    $.toTimestamp(120); // 02:00
    $.toTimestamp(98);  // 01:38
